## Folder Structure

|      Codebase      |            Description             |
| :----------------: | :--------------------------------: |
| [backend](backend) |           EduFi Backend            |

## Usage

### On Docker-Compose

```bash
docker-compose pull
docker-compose up
```

**The SQL setup script will be executed automatically at build time.**

Finally, open http://localhost:80 for the frontend client.
